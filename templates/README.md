# TermType Templates

## Directory Structure
- `easy/` - Simple sentences, basic vocabulary
- `medium/` - Sentences with numbers and basic data
- `hard/` - Complex sentences with punctuation
- `python/` - Python code examples
- `c_style/` - C, C++, Java, JavaScript examples
- `practice/` - Common typing mistakes
- `custom/` - Your own templates
