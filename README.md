# TermType - Terminal Typing Speed Test

A feature-rich, customizable typing speed test for the terminal.

## Features
- Multiple difficulty levels and categories
- Real-time WPM and accuracy tracking
- Color-coded feedback
- Custom template support
- Practice mode for common mistakes
- Code snippets for programmers
- Cross-platform (Linux, macOS, Windows)

## Installation

### From PyPI
```bash
pip install termtype
```

### From Source
```bash
git clone https://github.com/SAJAD-net/termtype
cd termtype
pip install -e .
```

### Usage
```bash
# Run the program
termtype

# Or as a module
python -m termtype
```

### Adding Custom Templates
1- Navigate to templates/custom/

2- Create a .txt file with your content

3- Use --- to separate multiple templates

4- Run TermType and select "Custom Templates"
